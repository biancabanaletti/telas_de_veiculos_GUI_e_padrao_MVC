Foi construída uma tela para visualizar os veículos cadastrados, permitindo selecionar um veículo, remover e adicionar um novo veículo.
Para a criação dessa interface gráfica foi utilizado o Tkinter, interligando as telas e juntando-as com as classes do Modelo já existente na Locadora de Veículos (motorhome e carro), aplicando o padrão MVC.
Criação do veiculo_list_view.py e Main.py para verificação e validação.
